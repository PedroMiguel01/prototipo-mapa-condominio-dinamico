
Este repositório apresenta um protótipo de mapa interativo desenvolvido com HTML, CSS, JavaScript, SVG e JSON.

O objetivo deste projeto é demonstrar a construção de uma interface visual simples, onde diferentes áreas de um mapa podem ser identificadas e exibidas de forma dinâmica no navegador.

## Objetivo do Protótipo

Este projeto foi desenvolvido como um exercício prático de front-end, com foco em:

- Estruturação de página com HTML
- Estilização com CSS
- Manipulação de elementos com JavaScript
- Uso de SVG para representação visual
- Leitura de dados fictícios a partir de JSON
- Organização inicial de um projeto web

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- SVG
- JSON

## Sobre os Dados

Todos os dados utilizados neste protótipo são fictícios e possuem finalidade exclusivamente demonstrativa.

Este projeto não contém informações reais, comerciais, confidenciais ou vinculadas a qualquer empresa, cliente ou operação.

## Como Visualizar

Para visualizar o protótipo:

1. Baixe ou clone este repositório.
2. Abra o arquivo `README.md` e siga as instruções.
3. Interaja com o mapa conforme os recursos disponíveis no protótipo.

## Status do Projeto

Este projeto está em fase de protótipo e foi publicado com o objetivo de demonstrar conhecimentos iniciais em desenvolvimento web e organização de interfaces interativas.
